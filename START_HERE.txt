📋 TO START THE APPLICATION:

1️⃣  OPEN FIRST TERMINAL (for Backend):
   cd backend
   npm run dev
   
   Wait for: "🚀 Dispatch Optimizer API running on http://localhost:3001"

2️⃣  OPEN SECOND TERMINAL (for Frontend):
   cd frontend
   npm run dev
   
   Wait for: Vite dev server URL (usually http://localhost:3000)

3️⃣  OPEN BROWSER:
   Navigate to: http://localhost:3000

💡 TROUBLESHOOTING:

❌ Backend won't start?
   → Make sure port 3001 is not in use
   → Check that you're in the backend/ directory

❌ Frontend won't start?
   → Make sure port 3000 is not in use
   → Check that you're in the frontend/ directory

❌ No data showing?
   → Ensure backend started first
   → Refresh the browser
